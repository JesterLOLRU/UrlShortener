# Сервис по сокращению ссылок

## Технический стек
* Java 11
* Spring Boot
* Spring Cloud
* Maven
* PostgreSql
* Liquibase
* Swagger

## Запуск
В папке проекта:
> mvn compile

> mvn package

> java -jar target/UrlShortener-1.0.jar


## Endpoints
Потыкать все можно здесь:
http://localhost:8080/swagger-ui.html
